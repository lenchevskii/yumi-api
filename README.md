## YUMI API ([documentation](https://yumi2.docs.apiary.io/#))

## Installation

```bash
$ npm install
```

## Creating `.env` in root folder:

```bash
$ cat .env
  PORT=...

  DB_TYPE=...
  DB_HOST=...
  DB_PORT=...
  DB_USERNAME=...
  DB_PSSWD=...
  DB_NAME=...
```

## Running the app

```bash
# development
$ npm run start

# production mode
$ npm run start:prod
```
